# upowerw
UPOWERW is a simple upower command wrapper.

The following status will be returned.

- Displays the remaining charge
- Whether charging is in progress

# Dependency

none

# Setup

```
# /usr/local/bin/ must have a pass
sh ./init.sh
```

